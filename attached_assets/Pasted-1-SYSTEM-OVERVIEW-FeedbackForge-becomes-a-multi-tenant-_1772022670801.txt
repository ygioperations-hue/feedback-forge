1️⃣ SYSTEM OVERVIEW

FeedbackForge becomes a multi-tenant SaaS with two roles:

👤 Customer (Workspace Owner)

Manages their own projects

Installs widget

Views feedback

Uses AI insights

Manages billing

Redeems LTD (if applicable)

They cannot see other customers or system data.

🛠 Platform Admin (You)

Manages all users

Generates LTD codes

Views global metrics

Can suspend accounts

Can manually upgrade users

Can view any customer project (read-only or support mode)

Only you have this access.

If valid:

plan_type = lifetime

ltd_code.is_used = true

ltd_code.used_by_user_id = user.id

Paywall lifted permanently.

7️⃣ STRIPE INTEGRATION (Multi-Tenant Correct)

Only customers use Stripe.

Platform admin does NOT subscribe.

Flow:

Customer clicks Subscribe

Stripe Checkout session created

Webhook updates:

plan_type = monthly OR yearly
stripe_customer_id saved

On subscription cancellation:

plan_type = none

Unless:

plan_type = lifetime

Never override lifetime.
